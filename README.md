
## 中文情绪识别

通过微博数据的表情，尝试用表情等信息标注文本数据，产生一定的情绪识别效果

细节待补充...

PS：一般情感识别是sentiment的翻译，此关键字的相关研究往往是两极、三极的，就是正负情绪，或者外加中性情绪。
而emotion一般指多种情绪，如开心、伤心、生气、反感、惊讶等。
